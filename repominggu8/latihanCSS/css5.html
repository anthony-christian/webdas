Saya sedang belajar mengenai tentang Cascade Style Sheet (CSS) di mata kuliah Pemrograman Berbasis Web Dasar ini.

Selektor adalah kata kunci dan/atau simbol yang digunakan pada Cascade Style Sheet (CSS) untuk menyeleksi atau memilih atribut elemen Hypertext Markup Language (HTML).

Selektor Class adalah selektor yang bertujuan untuk menyeleksi atau memilih atribut elemen Hypertext Markup Language (HTML) berdasarkan nama dari class yang diberikannya.

Selektor Class dibuat dengan menggunakan tanda titik pada bagian depannya.
