Daftar (List) HTML (Hypertext Markup Language)

Daftar (list) pada HTML memungkinkan pengembang web untuk mengelompokkan sekumpulan item terkait dalam daftar.

Daftar HTML Tidak Berurutan (An Unordered HTML List) :

  • Daftar (list) Pertama;
  • Daftar (list) Kedua;
  • Daftar (list) Ketiga;
  • dan seterusnya.

Daftar HTML Berurutan (An Ordered HTML List) :

  1. Daftar (list) Pertama;
  2. Daftar (list) Kedua;
  3. Daftar (list) Ketiga;
  4. dan seterusnya.
