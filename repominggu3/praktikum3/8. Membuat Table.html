Tabel (Table) HTML (Hypertext Markup Language)

Tabel (Table) HTML memungkinkan pengembang web untuk mengatur data menjadi baris (row) dan kolom (column).

Tag <table> mendefinisikan sebuah tabel (table) HTML.
Setiap baris tabel (table row) didefinisikan dengan tag <tr>.
Setiap tajuk tabel atau judul tabel (table header) didefinisikan dengan tag <th>.
Setiap sel tabel data (table data) didefinisikan dengan tag <td>.

Secara default, teks dalam tajuk tabel atau judul tabel (table header) yang didefinisikan dengan tag <th> elemen akan dicetak tebal (Bold) dan berada di posisi tengah (Center).
Secara default, teks dalam sel tabel data (table data) yang didefinisikan dengan tag <td> elemen akan beraturan dan dicetak di posisi rata kiri (Left).
Untuk mengatur perataan teks (text alignment) pada tabel, maka gunakan text-align properti dari CSS.

Untuk menambahkan garis pembatas ke dalam tabel, maka tambahkan border properti dari CSS.
Untuk membuat garis pembatas menjadi satu garis pembatas, maka gunakan border-collapse properti dari CSS.
Padding merupakan pengisi sel tabel data (table data) yang menentukan ruang antara konten sel tabel data (table data) dengan garis pembatasnya. Jika Anda tidak menentukan padding, maka sel tabel data (table data) akan ditampilkan tanpa adanya padding. Untuk mengatur dan menentukan padding, maka gunakan <padding> properti dari CSS.
Spasi perbatasan (border spacing) akan menentukan jarak antarsel tabel data (table data). Untuk mengatur spasi perbatasan (border spacing) pada tabel, maka gunakan border-spacing properti dari CSS.
Untuk membuat rentang sel tabel data (table data) menjadi lebih dari satu kolom, maka gunakan tag atribut colspan.
Untuk membuat rentang sel tabel data (table data) menjadi lebih dari satu baris, maka gunakan tag atribut rowspan.
Untuk menambahkan keterangan ke dalam tabel, maka gunakan tag atribut <caption>.
