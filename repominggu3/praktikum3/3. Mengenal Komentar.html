Komentar (Comments) HTML (Hypertext Markup Language)

Ini adalah sebuah paragraf HTML yang ditampilkan di browser.
Komentar HTML tidak ditampilkan di browser, akan tetapi dapat membantu mendokumentasikan kode sumber HTML Anda.
Dengan komentar HTML ini, Anda dapat menempatkan pemberitahuan dan pengingat di kode HTML Anda.
Komentar bisa juga digunakan untuk men-debug HTML, karena Anda dapat mengomentari baris kode HTML satu per satu untuk mencari kesalahan.
